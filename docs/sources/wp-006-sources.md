## Primary (fetched and verified)

- https://rdap.org/domain/<name> — accessed 2026-07-06 — RDAP bootstrap used for all gTLD registration checks (404 = no registration; 200 = registered). Verified for: sankofaehr/.com/.org/.africa (404), sankofahealth.com/.org (200), nkabomhealth.* (404), nkabom.com (200), apomuden.com (200), apomuden.org/.africa (404), akomahealth.com (200), akomahealth.org/.africa (404), ayaresa.com/.org (200), ayaresa.africa (404).
- DNS (`host`) lookups — accessed 2026-07-06 — NXDOMAIN vs resolution for the same list plus *.com.gh candidates (all NXDOMAIN; indicative only, no public .gh RDAP).
- https://registry.nic.gh — accessed 2026-07-06 — NIC Ghana registry system; live but redirects to `login.jsp` (registrar login, no public search).
- https://nic.gh/whois.php — accessed 2026-07-06 — NIC Ghana public WHOIS lookup form; live but reCAPTCHA-gated, cannot be scripted; manual .gh/.com.gh checks must go here.
- https://orc.gov.gh/ and https://orc.gov.gh/service/entity-information-search/ — accessed 2026-07-06 — Office of the Registrar of Companies; live; advertises Entity Information Search / Name Search; links e-portal at egovonline.gegov.gov.gh.
- https://egovonline.gegov.gov.gh/RGDPortalWeb/jsf/namesearch/NameSearch.xhtml — accessed 2026-07-06 — old RGD e-services name search; **unreachable (connection failure)** at access time.
- https://rgd.gov.gh/ — accessed 2026-07-06 — Registrar General's Department (Industrial Property/trademarks); live; no public online trademark database.
- https://api.github.com/users/Apomuden — accessed 2026-07-06 — GitHub org "Apomuden": 2 repos (`api` PHP, `webapp` "Web app for apomuden"), no bio/site; dormant.
